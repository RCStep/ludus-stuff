# Ludus Ranges

These directories contain Ludus Ranges.

For more information see: https://docs.ludus.cloud/docs/quick-start/deploy-range

You may import a range configuration with:
```
cd ranges
ludus range config set -f ludus-range-config.yml
```

Ranges:

| Range | Description |
| ---      | ---      |
| kitchen_sink | Ludus range configuration with AD DC, servers, and workstations. ADCS, Vuln-AD, MDE, Elastic, Sysmon, Splunk, Litterbox, kali, and vulhub server deployed |
