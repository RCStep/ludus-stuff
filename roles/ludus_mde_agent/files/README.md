### File Descriptions

- **Win10_11_GatewayWindowsDefenderATPOnboardingPackage.zip**: Onboarding file for Windows 10 and 11. "Win10_11_" prepended to the file name.
- **Win2019_2022_2025_GatewayWindowsDefenderATPOnboardingPackage.zip**: Onboarding file for Windows Server 2019, 2022, and 2025. "Win2019_2022_2025" prepended to the file name.
- **README.md**: This file, providing an overview of the folder contents.

### Usage Instructions

1. Download the appropriate onboarding package from the MDE security portal (Settings-Endpints-Device Management-Onboarding).
2. Prepend the approprate targeting description to the file name of the zip.
3. Place the files into this role's Files folder here.

### Notes

- Only the MDE onboarding files for Windows 10, 11, 2019, 2022, and 2025 are supported by this role for now.
- Refer to the role documentation for additional details.
